Toʻgʻri javob C

Savol

К телу приложены силы 69 Н и 74 Н. Если их равнодействующая равна 5 Н, определите угол между ними.

A) 90° B) 150° C) 180° D) 60°

Yechim

Ikki kuchning natijaviysi kosinuslar teoremasi bilan topiladi:

$$F^2 = F_1^2 + F_2^2 + 2F_1F_2\cos\theta$$

$$5^2 = 69^2 + 74^2 + 2\cdot 69\cdot 74\cdot\cos\theta$$

$$25 = 4761 + 5476 + 10212\cos\theta$$

$$25 = 10237 + 10212\cos\theta$$

$$\cos\theta = \frac{25 - 10237}{10212} = \frac{-10212}{10212} = -1$$

$$\theta = 180^\circ$$

(Bu natija tabiiy: kuchlar bir-biriga qarama-qarshi yo’nalgan bo’lsa, natijaviy kuch ularning ayirmasiga teng bo’ladi: $74 - 69 = 5$ N — berilgan qiymat bilan mos keladi.)

Javob: C
