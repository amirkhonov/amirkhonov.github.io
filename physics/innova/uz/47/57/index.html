Arximed kuchi

Toʻgʻri javob B

Savol

Quyidagi zichliklari berilgan jismlarning qaysilari suvda cho’kadi? 1) $\rho=700$ kg/m³; 2) $\rho=1200$ kg/m³; 3) $\rho=2$ g/cm³; 4) $\rho=0,2$ g/cm³; 5) $\rho=8900$ kg/m³.

A) 1, 2, 4. B) 2, 3, 5. C) 3, 4, 5. D) 2, 4, 5. E) 1, 3, 4.

Yechim

Jism suvda cho’kishi uchun uning zichligi suv zichligidan ($\rho_s = 1000$ kg/m³) katta bo’lishi kerak.

Har bir jism zichligini suv zichligiga solishtiramiz (kerakli birliklarga o’tkazamiz):

  1. $\rho_1 = 700$ kg/m³ $< 1000$ — cho’kmaydi (suzadi).

  2. $\rho_2 = 1200$ kg/m³ $> 1000$ — cho’kadi.

  3. $\rho_3 = 2$ g/cm³ $= 2000$ kg/m³ $> 1000$ — cho’kadi.

  4. $\rho_4 = 0,2$ g/cm³ $= 200$ kg/m³ $< 1000$ — cho’kmaydi (suzadi).

  5. $\rho_5 = 8900$ kg/m³ $> 1000$ — cho’kadi.

Demak, suvda cho’kadigan jismlar — 2, 3, 5.

Bu B javobiga mos keladi.

Javob: B
