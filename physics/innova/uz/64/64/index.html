Izojarayonlar

Toʻgʻri javob C

Savol

Hajmi 0,1 m³ bo’lgan silindrik idish yuzi 0,01 m² bo’lgan porshen bilan yopilgan. Gazning absolut temperaturasi izobarik ravishda 4 marta pasaytirilganda porshen idish tubidan necha metr balandlikda bo’ladi?

A) 1,52. B) 2. C) 2,5. D) 3. E) 4.

Yechim

Berilgan: $V_1 = 0,1$ m³, $S = 0,01$ m².

Boshlang’ich balandlik:

$$h_1 = \frac{V_1}{S} = \frac{0,1}{0,01} = 10 \text{ m}$$

Izobarik jarayonda harorat 4 marta pasaytirilgani uchun hajm ham 4 marta kamayadi:

$$V_2 = \frac{V_1}{4} = \frac{0,1}{4} = 0,025 \text{ m}^3$$

Yangi balandlik:

$$h_2 = \frac{V_2}{S} = \frac{0,025}{0,01} = 2,5 \text{ m}$$

Bu C javobiga mos keladi.

Javob: C
