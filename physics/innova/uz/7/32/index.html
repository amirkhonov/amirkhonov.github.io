32-masala

Tekis o'zgaruvchan harakatda ko'chish

Toʻgʻri javob A

Savol

Ushbu bog’lanishlarning qaysilari tekis o’zgaruvchan harakatni tavsiflaydi:

  1. $v = 4 + 3t$; 2) $s = 3 + 2t$; 3) $s = 3t - t^2$; 4) $v = 3t^2$; 5) $s = -3t$.

A) 1, 3. B) 1, 4. C) 2, 3. D) 3, 4. E) 1, 5.

Yechim

Tekis o’zgaruvchan harakat — tezlanish o’zgarmas bo’lgan harakat.

  1. $v=4+3t$ — chiziqli, $a=3$=const → tekis o’zgaruvchan.
  2. $s=3+2t$ — chiziqli koordinata, tezlik const, $a=0$ → tekis (bir jinsli) harakat, tezlanuvchan emas.
  3. $s=3t-t^2$ — kvadratik, $a=-2$=const → tekis o’zgaruvchan.
  4. $v=3t^2$ — tezlik $t^2$ ga proportsional, tezlanish $a=6t$ o’zgaruvchan → tekis o’zgaruvchan emas.
  5. $s=-3t$ — chiziqli, tezlik const, $a=0$ → tekis harakat.

Demak, 1 va 3 - tekis o’zgaruvchan harakat.

Javob: A
