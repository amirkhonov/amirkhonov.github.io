33-masala

Joul–Lens qonuni

Toʻgʻri javob C

Savol

Elektr isitkich bir xil $R$ qarshilikli uchta spiralga ega. Agar uchala spiral parallel ulansa, unga qo’yilgan idishdagi suv 6 minutda qaynaydi. Spirallar rasmda ko’rsatilgandek ulansa, shu suv necha minutda qaynaydi?

A) 24. B) 4. C) 12. D) 16. E) 6.

Yechim

Bitta $R$ qarshilik yolg’iz ulansa quvvati $P_1=\dfrac{U^2}{R}$ bo’ladi, kerakli issiqlik esa $Q$.

Uchta $R$ parallel ulanganda (barcha uch spiral bir-biriga parallel):

$$R_{parallel} = \frac{R}{3}, \qquad P_{parallel} = \frac{U^2}{R/3} = 3P_1$$

$$Q = P_{parallel}\cdot t_{parallel} = 3P_1\cdot 6$$

Rasmdagi ulanish: ikkita spiral ($2R$) ketma-ket bitta tarmoqni tashkil qiladi, uchinchi spiral ($R$) esa unga parallel ulangan.

Ekvivalent qarshilik:

$$R_{ekv} = \frac{2R\cdot R}{2R+R} = \frac{2R^2}{3R} = \frac{2R}{3}$$

Bu holdagi quvvat:

$$P_{yangi} = \frac{U^2}{R_{ekv}} = \frac{U^2}{2R/3} = \frac{3U^2}{2R} = 1,5P_1$$

Kerakli vaqt:

$$t_{yangi} = \frac{Q}{P_{yangi}} = \frac{3P_1\cdot 6}{1,5P_1} = \frac{18}{1,5} = 12 \text{ min}$$

Bu C javobiga mos keladi.

Javob: C
